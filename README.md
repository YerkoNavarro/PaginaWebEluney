# PaginaWebEluney

## Descripción
Plataforma web diseñada para un negocio de comida rápida. El sitio permite gestionar de forma externa el menú y sus costos mediante una conexión directa con Google Sheets.

## Vistas
* **Home:** Presentación del negocio.
* **Menú:** Listado de productos con precios dinámicos.
* **Quienes Somos:** Información institucional.
* **Contacto:** Datos de localización y formulario.

## Tecnologías
* React.js
* Bootstrap
* Google Sheets API
